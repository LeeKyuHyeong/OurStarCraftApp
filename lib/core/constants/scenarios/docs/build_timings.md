# 빌드별 표준 타이밍 (전 종족전 공용 레퍼런스)

> 시나리오 작성 시 참고할 실제 프로 경기 기준 타이밍 모음.
> 같은 빌드가 여러 종족전에 등장할 경우 이 파일을 먼저 확인해 일관된 타이밍을 사용한다.
> 기준: 스타크래프트 브루드워 Fastest 속도, 파이팅 스피릿급 표준 맵.

## 사용 규칙

1. **새 빌드 시나리오를 작성하기 전 이 파일을 먼저 확인**한다.
2. 이미 등재된 빌드라면 여기 타이밍을 그대로 사용 (시나리오 line 번호도 동일하게 매핑).
3. 새 빌드를 조사했다면 **즉시 이 파일에 추가**해 다음 작업자가 중복 조사하지 않도록 한다.
4. 빌드 ID는 종족전 prefix를 제외한 빌드명 기준으로 그룹화 (예: `4pool`은 ZvZ/ZvT/ZvP 공용).
5. 시나리오 line 매핑 표준은 본 문서 하단 **"Line 매핑 가이드"** 참조.

## 저그 빌드

### ZvZ 공통 특성 (라바 운용)

ZvZ는 매 라바를 **드론 / 저글링 / 뮤탈 / 스커지** 중 어디에 쓰느냐로 양상이 갈리는 종족전이다. 모든 ZvZ 빌드는 이 트레이드오프 위에서 설계됨:

- **공격 유닛(저글링/뮤탈)에 라바를 더 쓰는 빌드** (4풀, 9풀 발업, 9풀 레어): 초반 압박은 강하지만 드론·해처리가 뒤처져 장기전 자원 부족
- **드론·해처리에 라바를 더 쓰는 빌드** (12풀, 12앞): 라바 회전이 두 배로 늘어 후반이 강하지만 초반에 공격 유닛이 부족해 버티기가 어려움
- 그래서 9풀 레어처럼 빠른 테크 빌드는 **앞마당/본진 추가 해처리를 쉽게 올리지 못함** — 라바 자원이 스파이어/뮤탈에 집중되어야 성립
- 12해처리는 성공 시 라바 운용을 2배로 가져갈 수 있다는 점에서 강력하지만, 초반 수비를 노발업 저글링+성큰으로 견뎌내야 함

### 원팩원스타 / 원팩투스타 (1Fac 1Starport / 1Fac 2Starport)

- **사용 종족전**: TvT (`tvt_1fac_1star`, `tvt_1fac_2star`)

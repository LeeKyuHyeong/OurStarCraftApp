# 시나리오 원가 기준표

시나리오 ScriptEvent의 `homeArmy`, `awayArmy`, `homeResource`, `awayResource` 값을 결정하는 기준.
에이전트 지시 시 이 파일을 참조하도록 하면 됨.

---

## 공통 규칙

### 병력 (army)
- **병력 = 서플라이** (유닛의 인구수 소모량)
- 저글링 2마리 = 서플라이 1 = 병력 1
- 시즈탱크 1대 = 서플라이 2 = 병력 2
- 드론/SCV/프로브 = 병력 변동 없음 (일꾼은 army에 미포함)
- 건물 건설 = 병력 변동 없음
- 변태 유닛(러커, 가디언 등): 변태 전 유닛의 서플라이 변동분만 적용

### 자원 (resource)
- **자원 = 미네랄 + 가스 합산**
- 건물도 동일 (스포닝풀 200광 = 자원 200)
- 변태 유닛: **변태 비용만** (원본 유닛 비용은 생산 시 이미 지불)

### recovery
- base: **50/line** (전 시나리오 공통, 개별 설정 금지 — D1 규칙)
- 확장당: **+100** 추가 (homeExpansion/awayExpansion 플래그)
- 자원 변동은 recovery 대비 유의미해야 함 (최소 50 이상)

### 이벤트 자원 부호
- 생산/건설: **음수** (자원 소비)
- 드론 축적: **양수** (경제 성장)
- 견제 피해: 상대측 **음수** (드론 수 × 50~100)
- decisive: 보통 0/0

## 이벤트 패턴 예시

### 생산 이벤트
```dart
// 저글링 6기 생산
homeArmy: 3, homeResource: -150,  // 서플라이 3, 50×3라바

// 뮤탈 2기 생산
homeArmy: 4, homeResource: -400,  // 서플라이 4, 200×2

// 마린 4기 + 메딕 1기
homeArmy: 5, homeResource: -275,  // 서플라이 5, 50×4+75×1

// 드라군 3기
homeArmy: 6, homeResource: -525,  // 서플라이 6, 175×3

// 시즈탱크 2대
homeArmy: 4, homeResource: -500,  // 서플라이 4, 250×2
```

### 건물 건설
```dart
// 팩토리 + 머신샵
homeResource: -400,  // 300+100

// 게이트웨이 2개
homeResource: -300,  // 150×2

// 스파이어
homeResource: -350,  // 200+150
```

### 견제/교전
```dart
// 드론 2기 견제 사망
awayResource: -100,  // 드론 2×50

// 저글링 교전 (홈 4마리 사망, 어웨이 6마리 사망)
homeArmy: -2, awayArmy: -3,
homeResource: -100, awayResource: -150,  // 50×쌍수

// 마린 3기 사망
awayArmy: -3, awayResource: -150,  // 50×3
```

### 묶음 이벤트
```dart
// 스파이어(350) + 뮤탈 2기(400) 동시
homeArmy: 4, homeResource: -750,

// 팩토리(300) + 시즈탱크(250) 동시
homeArmy: 2, homeResource: -550,
```

## 에이전트 지시용

시나리오 보정 시 에이전트에게 아래를 지시:

> `lib/core/constants/scenarios/docs/unit_costs.md` 참조하여 모든 ScriptEvent의 army/resource 값을 원가 기준표에 맞게 수정하라. 텍스트와 구조는 변경 금지.

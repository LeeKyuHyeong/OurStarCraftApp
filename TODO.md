#### 시스템 구조 메모
- **조건 반영**: 맵정보(`MapRequirement`/`requiresMapTag`) + 능력치(`favorsStat`) + 빌드선택(스크립트 매칭) + 스나이핑(빌드 선택 단계)
- **다양성**: skipChance(이벤트 스킵) + altText(대체 텍스트) + ScriptBranch(능력치 분기) + _transformEnding(방송 어미)
- **분기 선택**: 적격 필터(conditionStat 비교) → 가중 랜덤(baseProbability) → conditionStat 없으면 항상 적격
- **폴백**: 스크립트 매칭 안되면 기존 빌드스텝+클래시 시스템 100% 유지
- **파일**: `scenario_scripts.dart`(클래스+데이터) + `scenario_tvz.dart`(TvZ part파일) / `match_simulation_service.dart`(엔진)

## 1. 게임 개요

### 1.1 컨셉
- 스타크래프트 프로게이머를 직접 플레이하는 게임이 아님
- 선수 및 팀 육성에 집중
- 경기는 시뮬레이션으로 진행 (텍스트 로그 + 자원/병력 변화)

### 1.2 핵심 기능
- 선수 관리 (스카우트, 육성, 방출)
- 팀 운영
- 시즌 진행 (정규 시즌, 토너먼트, 위너스리그)
- 경기 시뮬레이션 (x1, x2, x4, x8 배속)

### 1.3 레퍼런스
- MyStarcraft (cafe.naver.com/kaiknight)

## 3. 선수 시스템

### 3.1 기본 정보
```
- 이름
- 사진 (유저가 직접 등록)
- 종족 (T: 테란, Z: 저그, P: 프로토스)
- 소속팀
- 레벨 (1~10)
- 등급 (F- ~ SSS)
- 컨디션 (0~150%)
```

### 3.2 능력치 (8가지)

| 능력치 | 설명 | 게임 내 영향 |
|--------|------|--------------|
| **센스** | 상대 정보 파악 | 상대 빌드 파악, 기습 회피 |
| **컨트롤** | 유닛 조작 능력 | 전투 효율, 병력 손실 감소, 마이크로 이벤트 |
| **공격력** | 공격 플레이 | 러쉬 확률, 공격 피해량, 올인 성공률 |
| **견제** | 멀티태스킹/견제 | 드랍, 멀티 견제, 일꾼 학살 |
| **전략** | 빌드/전술 | 특수 전략, 상성 극복, 크리티컬 이벤트 |
| **물량** | 자원/멀티 관리 | 자원 수급, 확장 성공률, 후반 물량 |
| **수비력** | 방어 플레이 | 러쉬 방어, 드랍 방어, 피해 감소 |
| **정찰** | 정찰 능력 | 타이밍 파악, 카운터 가능 |

### 3.3 능력치 범위
- 각 능력치: 0 ~ 999
- 8개 합계: 0 ~ 7992

### 3.4 컨디션 시스템
- 기본: 100%
- 폼댄 (좋음): 100% 초과 (최대 150%), 초록색 표시
- 짜증 (나쁨): 100% 미만, 능력치 하락
- 실제 능력치 = 기본 능력치 × (컨디션% / 100)

## 5. 레벨 시스템

### 5.1 레벨 범위
- 레벨 1 ~ 10 (선수의 경력/나이)

### 5.2 레벨별 의미

| 레벨 | 시기 | 성장력 | 하락 위험 |
|------|------|--------|-----------|
| **1** | 신인 1년차 | 최고 | 없음 |
| **2** | 신인 2년차 | 매우 높음 | 없음 |
| **3** | 신인 3년차 | 높음 | 거의 없음 |
| **4** | 전성기 초입 | 보통 | 없음 |
| **5** | 전성기 | 보통 | 낮음 |
| **6** | 전성기 | 낮음 | 낮음 |
| **7** | 전성기 후반 | 낮음 | 보통 |
| **8** | 베테랑 | 매우 낮음 | 보통 |
| **9** | 노장 | 거의 없음 | 높음 |
| **10** | 은퇴 임박 | 없음 | 매우 높음 |

### 5.3 레벨 구간

| 구간 | 레벨 | 특징 |
|------|------|------|
| 신인 | 1, 2, 3 | 성장 잠재력 최고 |
| 전성기 | 4, 5, 6, 7 | 최고 기량 발휘 |
| 베테랑/노장 | 8, 9, 10 | 하락 시작 |

### 5.4 레벨업 시스템
- **기본**: 2시즌마다 레벨 +1
- **랜덤 요소**: 시즌 종료 시 10~20% 확률로 추가 +1
- **평균 은퇴 시점**: 약 25시즌 전후
- **적용 범위**: 8팀 전체 선수 동일 적용

```
예시 (레벨1 신인 입단 시)
시즌 2 종료 → 레벨 2
시즌 4 종료 → 레벨 3
...
시즌 18 종료 → 레벨 10 (기본)
+ 랜덤 발동 시 더 빨리 은퇴 가능
```

## 6. 성장/하락 시스템

### 6.1 핵심 원칙
1. **레벨이 낮을수록** 성장폭이 큼
2. **낮은 능력치**가 더 많이 성장
3. **레벨이 높으면** 이겨도 하락 가능 (노화)
4. **강한 상대에게 패배**해도 신인은 성장

### 6.2 승리 시 변화

| 레벨 | 변화량 | 설명 |
|------|--------|------|
| 1 | +20 ~ +40 | 크게 성장 |
| 2 | +15 ~ +35 | 매우 높음 |
| 3 | +10 ~ +25 | 높음 |
| 4~5 | +5 ~ +15 | 보통 |
| 6~7 | +0 ~ +10 | 소폭 |
| 8 | -5 ~ +5 | 유지 or 소폭 하락 |
| 9 | -10 ~ +3 | 이겨도 하락 가능 |
| 10 | -15 ~ 0 | 이겨도 하락 |

### 6.3 패배 시 변화

| 레벨 | 변화량 | 설명 |
|------|--------|------|
| 1 | +5 ~ +25 | 그래도 성장 (강한 상대 +30) |
| 2 | +0 ~ +20 | 성장 가능 |
| 3 | -5 ~ +15 | 소폭 성장 가능 |
| 4~5 | -10 ~ +5 | 유지 or 소폭 하락 |
| 6~7 | -15 ~ 0 | 하락 |
| 8 | -20 ~ -5 | 확실히 하락 |
| 9 | -25 ~ -10 | 큰 하락 |
| 10 | -35 ~ -15 | 매우 큰 하락 |

### 6.4 능력치별 성장 확률 (낮을수록 잘 오름)

| 능력치 범위 | 성장 확률 | 성장량 |
|-------------|-----------|--------|
| 0 ~ 300 | 90% | +15 ~ +30 |
| 301 ~ 500 | 70% | +10 ~ +20 |
| 501 ~ 700 | 50% | +5 ~ +15 |
| 701 ~ 850 | 30% | +2 ~ +8 |
| 851 ~ 999 | 10% | +0 ~ +3 |

### 6.5 선수 가치 평가

| 유형 | 조건 | 평가 |
|------|------|------|
| **최고 영입** | 레벨 1~2, A-~S급 | ⭐⭐⭐⭐⭐ |
| **좋은 영입** | 레벨 2~3, B+~A+급 | ⭐⭐⭐⭐ |
| **즉시 전력** | 레벨 4~6, S~SS+급 | ⭐⭐⭐ |
| **단기 활용** | 레벨 7~8, S+이상 | ⭐⭐ |
| **영입 비추** | 레벨 8+, A이하 | ⭐ |
| **레전드** | 레벨 9~10, SS이상 | ⭐⭐⭐ (명성) |

## 7. 팀 관리 시스템

### 7.1 행동력 시스템
- **획득**: 2매치 진행 시 행동력 100 획득
- **용도**: 팀 관리 활동에 소모
- **상한**: 미정 (추후 결정)

### 7.2 선수 행동 UI

```
┌──────────────────────────────────────────────────────────────┐
│                       선수 행동                              │
├─────────────────────────┬────────────────────────────────────┤
│                         │  [1.휴식]  [2.특훈]  [3.팬미팅]    │
│   MY TEAM PLAYER        ├────────────────────────────────────┤
│                         │                                    │
│  선수들    행동력 컨디션 │      휴식을 취합니다               │
│  고강민(Z)   0    100   │                                    │
│  권태일(P)   0    100   │      소모 행동력 = 50              │
│  권성대(Z)   0    100   │                                    │
│  ...                    │      쉬면서 컨디션을 회복합니다.   │
│                         │                                    │
├─────────────────────────┴────────────────────────────────────┤
│      [EXIT (Bar)]                        [실행 (C)]          │
└──────────────────────────────────────────────────────────────┘
```

### 7.3 팀 관리 메뉴

| 활동 | 행동력 소모 | 효과 | 컨디션 변화 |
|------|-------------|------|-------------|
| **휴식** | 50 | 컨디션 회복 (+4~5) | +4~5 |
| **특훈** | 100 | 능력치 랜덤 상승 (레벨에 따라 상승폭 다름) | -1 |
| **팬미팅** | 200 | 치어풀 랜덤 획득, 소지금 획득 (등급 비례) | -2 |

### 7.4 특훈 시스템
- **효과**: 선수 능력치 랜덤 상승
- **레벨 영향**: 낮은 레벨일수록 상승폭 큼
- **컨디션**: -1

### 7.5 팬미팅 시스템
- **치어풀 획득**: 일정 확률로 획득 (등급 영향 여부 설정 가능)
- **소지금 획득**: 참여 선수 등급에 비례하여 금액 증가
- **컨디션**: -2

### 7.6 컨디션 회복 타이밍

## 10. 이적 시장

### 10.1 이적 시장 오픈 시점
- **시즌 종료 후** 이적 시장 오픈
- 무소속 선수 조회 + 타팀 선수 트레이드 가능

### 10.2 이적 유형

| 유형 | 방식 | 예시 |
|------|------|------|
| **무소속 영입** | 몸값만 지불 | SS급 레벨3 → 780만원 |
| **팀간 트레이드** | 우리 선수 + 차액 | SS급 ↔ A+급 + 233만원 |

### 10.3 몸값 계산 공식

```
최종 몸값 = 기본 몸값(등급) × 레벨 계수
```

#### 등급별 기본 몸값

| 등급 | 기본 몸값 |
|------|-----------|
| SSS | 1000만원 |
| SS+ | 800만원 |
| SS | 600만원 |
| SS- | 500만원 |
| S+ | 400만원 |
| S | 300만원 |
| S- | 250만원 |
| A+ | 200만원 |
| A | 150만원 |
| A- | 100만원 |
| B+ | 70만원 |
| B | 50만원 |
| B- | 35만원 |
| C+ 이하 | 20만원 |

#### 레벨별 계수 (성장 잠재력 반영)

| 레벨 | 계수 | 이유 |
|------|------|------|
| 1 | ×1.8 | 성장 잠재력 최고 |
| 2 | ×1.5 | |
| 3 | ×1.3 | |
| 4~5 | ×1.0 | 전성기 기준 |
| 6~7 | ×0.8 | |
| 8 | ×0.5 | |
| 9 | ×0.3 | |
| 10 | ×0.1 | 은퇴 임박 |

#### 몸값 예시
- SS급 레벨2: 600만 × 1.5 = **900만원**
- SS급 레벨8: 600만 × 0.5 = **300만원**
- A+급 레벨1: 200만 × 1.8 = **360만원**

### 10.4 트레이드 계산

```
필요 차액 = 상대 선수 몸값 - 우리 선수 몸값
```

**예시:**
- 상대 SS급 레벨3 (780만) ↔ 우리 A+급 레벨5 (200만)
- 필요 차액: 780만 - 200만 = **580만원 추가 지불**

### 10.5 이적 시장 UI 흐름

```
[시즌 종료]
    ↓
[이적 시장 오픈]
├── [무소속 선수] → 랜덤 N명 조회 → 몸값 확인 → 영입
└── [상대팀 조회] → 선수 클릭 → 우리 선수 선택 → 차액 확인 → 트레이드
    ↓
[다음 시즌 시작]
```

### 10.6 무소속 선수 풀
- 시즌 종료 시 랜덤 생성
- 은퇴 선수 제외, 신인 포함
- 등급/레벨 다양하게 분포

### 11.11 능력치가 경기에 미치는 영향

#### 빌드 선택
```
공격형 = (공격력 + 컨트롤) / 2 → 러쉬, 올인 빌드
수비형 = (수비력 + 운영) / 2 → 안정 운영 빌드
전략형 = (전략 + 정찰) / 2 → 특수 빌드
```

#### 경기 단계별

| 단계 | 시간 | 주요 능력치 |
|------|------|-------------|
| 초반 | 0~3분 | 공격력(러쉬), 정찰(감지), 수비력(방어) |
| 중반 | 3~10분 | 컨트롤(견제), 운영(확장), 전략(테크) |
| 후반 | 10분~ | 운영(물량), 컨트롤(전투), 전략(역전) |

#### 전투 계산
```
공격점수 = (공격력×0.3) + (컨트롤×0.3) + (전략×0.2) + (랜덤×0.2)
방어점수 = (수비력×0.3) + (컨트롤×0.3) + (운영×0.2) + (랜덤×0.2)

승자 병력 손실: 10~30%
패자 병력 손실: 40~80%
```

**구조**
- 총 24개 조 (1조 ~ 24조)
- 각 조 4명 참가 (토너먼트 형식)
- 조별 Winner → 듀얼토너먼트 진출

**UI 구성**

**진행 방식**
- 텍스트 시뮬레이션 **없음** (결과만 표시)

- 이긴 선수가 대진표 위로 올라감
- 최종 Winner 결정 → 듀얼토너먼트 진출

**참가 선수**
- 우리팀 선수 약 6명 (각기 다른 조에 배정)
- 나머지는 타팀 선수 + 아마추어

### 15.5 개인리그 시뮬레이션
- **8강 전**: 텍스트 없이 승/패 결과만 표시
- **8강 이후**: 텍스트 전투 시뮬레이션 진행
- **승률 계산**: (미정) 능력치 차이 기반 확률 or 내부 텍스트 시뮬레이션 후 결과만 표시

### 15.6 시즌 종료 이벤트
- 2시즌마다 모든 선수 레벨 +1 (10~20% 확률로 추가 +1)
- 신인: 랜덤 능력치 +5~15
- 베테랑: 랜덤 능력치 -5~15
- 레벨 10 선수 은퇴 체크
- 컨디션 100% 리셋
- 신인 선수 3~5명 생성
- 이적 시장 오픈

## 16. 특수 이벤트

### 16.1 각성
- **조건**: 레벨 1~3, 상위 등급 상대 승리
- **확률**: 8%
- **효과**: 모든 능력치 +30~50
- **텍스트**: "{player} 선수, 각성했습니다!"

### 16.2 슬럼프
- **조건**: 3연패 이상
- **확률**: 30%
- **효과**: 컨디션 -15%, 능력치 일시적 -20
- **회복**: 2승 시 해제
- **텍스트**: "{player} 선수, 슬럼프에 빠졌습니다..."

### 16.3 부상
- **조건**: 경기당 3% 확률
- **효과**: 1~3경기 출전 불가
- **텍스트**: "{player} 선수, 부상으로 이탈!"

### 16.4 은퇴
- **조건**: 레벨 10 + 패배 후 등급 2단계 이상 하락
- **효과**: 선수 삭제
- **텍스트**: "{player} 선수가 은퇴를 선언했습니다."

### 16.5 컴백
- **조건**: 슬럼프 상태에서 승리
- **확률**: 20%
- **효과**: 슬럼프 해제, 컨디션 110%
- **텍스트**: "{player} 선수, 화려하게 돌아왔습니다!"

### 16.6 최고의 경기
- **조건**: 전성기(레벨 4~6) + 승리
- **확률**: 5%
- **효과**: 경기 중 능력치 +50, 경기 후 +10~20 성장
- **텍스트**: "{player} 선수, 오늘 최고의 컨디션입니다!"
